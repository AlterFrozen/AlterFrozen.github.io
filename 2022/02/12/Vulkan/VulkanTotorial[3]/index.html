Development Environment

Windows Platform

  • Visual Studio 2017 or 2019
  • Complete C++17 support

Vulkan SDK

Intro

You’ll need for developing Vulkan app via SDK which includes headers + standard validation layers + debugging tools + a loader for Vulkan functions

  • Loader：looks up the functions in the runtime similarly to GLEW for OpenGL

Downdown

  • Downdowing the SDK in LunarG website

Verify

  • Go to the directory where you installed the SDK open Bin directory and run the vkcube.exe demo you will see the following

If you receive an error message then ensure that your drivers are up-to-date,

include the Vulkan runtime and that your graphics card is supported. See the

introduction chapter for links to drivers from the major vendors.

Vital Directories

  • Bin：contains Vulkan loader and validation layers
  • Include：contains Vulkan headers
  • Lib：contains the libraries

Feel free to explore the other files , but we won’t need them for this tutorial

GLFW

  • 为了cross-platform需要使用GLFW创建窗口（GLFW也提供其他跨平台功能）

To benefitfrom the cross-platform advantages of Vulkan and to avoid the horrors of Win32,we’ll use the GLFW library to create a window, which supports Windows, Linux and MacOS. There are other libraries available for this purpose, like SDL, but the advantage of GLFW is that it also abstracts away some of the other platform specific things in Vulkan besides just window creation.

Download and Install

  • You can find the latest release of GLFW on the offfficial website.

  • 如果选择32位，就要链接Lib32目录，Lib目录是64位的！

  • （由于之前使用OpenGL已经配置好环境了，因此不演示具体细节，可以参考下图官方文档中给出的目录结构）

GLM

Intro

  • Vulkan does not include a library for linear algebra (unlike DirectX 12)
  • GLM is a nice library that is designed for use with graphics APIs and is also commonly used with OpenGL.
  • GLM is a header-only library, so just download the latest version and store it in a convenient location.

Setting up Visual Studio

Built Project

  • Create “Windows Desktop Wizard”

  • “Console Application(.exe)”（确保可以用console输出信息）

Project Properties

  • Open the project properties dialog and ensure that All Configurations is selected，, because most of the settings apply to both Debug and Release mode.
